Sustainable Meal Planner

The aim is to have a system that can help people automatically generate a sustainable meal plan for anywhere in the world. The system will need to be able to take into account the local climate, culture and cuisine. It will also need to be able to find recipes that use local ingredients and that are appropriate for the time of year.
